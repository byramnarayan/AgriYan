================================================================================
API ENDPOINTS
================================================================================

Authentication:
  POST /api/auth/login              - Login
  POST /api/auth/register           - Register

Plants:
  POST /api/plants/detect           - Identify plants
  GET  /api/plants/history          - Detection history

Farms:
  POST /api/farms/create            - Create farm
  GET  /api/farms/{id}              - Get farm details
  PUT  /api/farms/{id}              - Update farm
  DELETE /api/farms/{id}            - Delete farm

Crops:
  GET  /api/crops/recommend         - Get recommendations
  POST /api/crops/plant             - Log crop

Gamification:
  GET  /api/gamification/profile    - User profile
  GET  /api/gamification/leaderboard - Leaderboard

Alerts:
  GET  /api/alerts                  - Get alerts
  POST /api/alerts/acknowledge      - Mark as read

Dashboard:
  GET  /api/dashboard               - Dashboard view
  GET  /api/dashboard/stats         - Statistics

TROUBLESHOOTING
================================================================================

❌ "Phone number not recognized"
   → Check format: Must start with +91 or be 10 digits
   → Try: +919876543210 (exactly as shown)

❌ "Incorrect password"
   → Password is case-sensitive: Demo@123 (not demo@123)
   → Check Caps Lock is OFF

❌ "User already exists"
   → Run: rm agritech.db
   → Then: python seed_demo_user.py

❌ "Database locked"
   → Stop the server
   → Try again

❌ "Module not found"
   → Run: pip install -r requirements.txt
